# 综测笺

> 综合素质测评活动记录软件 —— 快速记录、自动计分、本地优先。

**综测笺** 是一款跨平台（Android / iOS / Windows）的综合素质测评（综测）活动记录工具。它把"记一笔活动"这件事压缩到几秒钟：选好学年和分类、填上活动名称，分数由你预先配置的计分模板自动算出。所有数据都保存在本地设备上，云端仅用于备份与跨设备同步。

---

## ✨ 特性

- **极速记录** —— 半屏录入面板专为单手操作设计，默认落入当前学年，填名即存。
- **差异化计分** —— 按「分类 / 是否获奖 / 获奖级别 / 名次 / 角色」配置规则，最具体的规则优先匹配，分数自动结算。
- **学年归档** —— 学年作为一级分类、活动类型作为二级分类；往届学年可一键归档，界面始终清爽。
- **可视化模板编辑** —— 分类、颜色、学年上限、获奖级别、名次、角色全部由你在模板中自定义，并可为每项配置填写提示。
- **附件管理** —— 支持为活动添加图片、PDF、Word、Excel、ZIP 等佐证材料。
- **导入 / 导出** —— 模板与全量数据均可导出为 JSON 文件，方便备份、迁移或共享；记录还可导出为 Excel 表单。
- **WebDAV 备份** —— 连接你自己的 WebDAV 服务器，把数据安全地同步到云端。
- **本地优先** —— 数据归你所有，离线可用，云端只是可选的备份层。

## 📐 计分模型

每条记录的分数由模板中的**计分规则列表**解析得出：一条规则只要它声明的每个字段（分类 / 是否获奖 / 获奖级别 / 名次 / 角色）都与记录匹配即生效，**最具体的规则胜出**。

记录卡片上的分数格式为 `当前累计(本次变动)/分类学年上限`，例如：

```
8(+3)/25
```

含义是：算上本次活动，该分类在本学年累计得 8 分，其中本次活动 +3 分，该分类学年上限为 25 分。

- 未分类的记录恒计 0 分。
- 学年总分 = 各分类得分（**按各自的分类上限封顶**）之和 + 未分类得分（不封顶）。
- 学年总分可以合法地超出学年上限（= 各分类上限之和），超出时以红色提示。

## 🧭 应用结构

应用由底部三个标签页组成：

| 页面 | 作用 |
| --- | --- |
| **记录** | 按「学年 → 活动类型」浏览与录入活动。点击 `+` 弹出半屏录入面板；左滑学年可归档，左滑记录可编辑 / 删除。 |
| **归档** | 查看已归档学年的记录（只读），左滑可取消归档。 |
| **设置** | 模板的导入 / 导出与可视化编辑、WebDAV 备份配置、关于信息。 |

## 🛠️ 技术栈

- **Flutter** —— 一套代码覆盖 Android、iOS、Windows。
- **状态管理** —— `provider` + `ChangeNotifier`，`AppState` 为模板、学年、记录的单一数据源，所有改动即时持久化。
- **本地存储** —— 原生平台写入应用目录下的 `template.json` / `data.json`（附件存于 `attachments/`）；Web 端降级到 `SharedPreferences`。
- **云端同步** —— 通过 `webdav_client` 实现 WebDAV 备份。

## 🚀 快速开始

确保已安装 [Flutter SDK](https://docs.flutter.dev/get-started/install)（要求 Dart SDK `^3.11.4`）。

```bash
flutter pub get          # 安装依赖
flutter run -d chrome    # 最快的预览方式（Web）
flutter run -d windows   # 原生 Windows（需 Visual Studio 的 C++ 桌面开发工作负载）
```

构建发布版：

```bash
flutter build apk        # Android
flutter build windows    # Windows
flutter build web        # Web，产物在 build/web/
```

### 平台说明
- **Windows 桌面**：需要开启「开发者模式」（用于插件符号链接）并安装带 C++ 工作负载的 Visual Studio。
- **Web**：无需额外工具链，是最快的预览方式；但 `path_provider` 与附件 / WebDAV 相关能力在 Web 上不可用（存储降级为 `SharedPreferences`，附件与 WebDAV 退化为空操作）。
