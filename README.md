# 综测活动记录软件

## 技术栈
使用Flutter进行跨平台开发，使用本地优先架构，云端只负责备份、跨平台同步功能

### 注意事项
模板应当选择合适的存储格式

## 特性
- 活动记录
- 活动分类
- 差异化计分
- 导入/导出分类模板
- 导入/导出数据
- WebDAV备份
- 附件支持图片、pdf、doc、xlsx、zip格式

## 功能详解
### 平台
安卓、iOS、Windows跨平台开发，其中Windows端既充当客户端（client）又充当服务端（server）

Windows端充当WebDAV服务器，在软件运行时开设一个WebDAV服务

### 核心逻辑
学年、活动分类、不同分类、不同获奖级别、角色对应的分数均由用户提前自行在模板中进行设置，软件只负责快速记录

允许模板作者为每个表项设置提示信息

### 应用页面
总体分三个页面：

- 记录
- 归档
- 设置

#### 记录页面

活动记录通过学年-活动类型进行分类，学年为一级分类，活动类型为学年下的二级分类。

收起状态下，左滑学年可以选择归档。归档的学年在该页面中不再显示，将在“归档”页面中显示

一级分类展示如下信息：
[学年， 当前分数/学年分数上限]

当前分数允许超出学年分数上限

点击学年可以展开/收起。展开时，在学年下方展示该学年的活动记录，不同活动类型用不同颜色标注（该颜色由用户在模板中指定），并在最上方标注出颜色对应的分类。点击某个分类时，可只显示该分类下的活动。
最新的学年以及最新添加的活动展示在最上方

点击+号，弹出记录窗口（半屏设计，方便单手操作），用户即可开始添加活动记录，默认添加至设置的当前学年下，且未分类。

在记录窗口中，用户需填写活动名称（必填），可以选择学年（仅可选择未归档的学年）、活动分类、是否获奖（默认未获奖）。
对于“是否获奖”，如果选择是，则让用户选择获奖等级（国家级、省级、校级、院级等）以及名次（一等奖、二等奖、三等奖等），（如果模板中规定对不同角色进行区分）并让用户选择是否是团队荣誉（默认为否），如果是团队荣誉，则让用户选择自己的角色（队长、第一队员、第二队员等）。
允许用户添加附件

对于活动的记录项，显示是否有附件（用图标表示）、活动名称、添加日期、分数（展示形式见下），单击可以查看详情（以卡片形式弹出），左滑可以选择删除、编辑。
对于分数的展示格式为：当前总分（变动）/该类活动学年总分，例如：8(+3)/25，意思是，算上该活动，该类活动一共得8分，此次活动加3分，学年上限为25分。

#### 归档页面

该页面展示用户归档了的学期，可以查看学期下的活动记录（逻辑与记录页面相同），不能进行添加和删除操作，左滑可以选择取消归档

#### 设置页面

##### 模板设置
